## Interaction Patterns

### Keyboard Shortcuts
- **Tab**: Indent in editor
- **Ctrl+S**: Save reminder
- **Enter**: Submit tag/keyword/category input
- **Escape**: Close modals (future)

### Mouse Interactions
- **Hover**: Background color change on buttons, table rows, cards
- **Click**: Button press effect (slight scale down)
- **Double Click**: Quick edit (future enhancement)
- **Drag**: Reorder items (future enhancement)

### Touch Interactions (Mobile)
- **Tap**: All button interactions
- **Scroll**: Horizontal scroll on status tabs
- **Swipe**: Navigate between pages (future)
