## Test Scenarios

### Scenario 1: Render Error (Caught ‚úÖ)

```typescript
// Component throws during render
if (shouldThrow) {
  throw new Error('Test error');
}

// Result:
// ‚úÖ Error Boundary catches it
// ‚úÖ Fallback UI shown
// ‚úÖ App doesn't crash
```

### Scenario 2: Async Error (NOT Caught ‚ùå)

```typescript
// Async operation throws

  await Promise.resolve();
  throw new Error('Async error');
};

// Result:
// ‚ùå Error Boundary does NOT catch it
// ‚ùå Unhandled promise rejection
// ‚ÑπÔ∏è  Use try/catch in the function
```
